## 🌟 Vision & Mission
> "To create a global financial infrastructure where **trust is visible, not blind**."

PathGuard eliminates the "black box" of modern finance. Every dollar has a verifiable passport, showing exactly where it travels, who handles it, and its real-time security status.

---

## 🧠 AI-Agent Intelligence (PathGuard Sentinel)
Our autonomous AI monitors every hop in the transaction chain:
- **Real-time Risk Scoring**: Instant detection of smurfing, layering, and phishing.
- **Natural Language Explanations**: "Your transfer is safe. It took 3 validated hops through Tier-1 banks."
- **Self-Healing Routes**: Automatically suggests alternatives during bank outages or fraud spikes.

## 🏗️ System Architecture & Tech Stack
Built for scale, speed, and absolute integrity.

| Layer | Technology | Purpose |
| :--- | :--- | :--- |
| **Ingestion** | Apache Kafka | High-throughput transaction streaming |
| **Intelligence** | Graph Neural Networks | Path risk & anomaly prediction |
| **Frontend** | Next.js / Flutter | Cinematic investigative workspace |
| **Ledger** | Merkle Chaining | Immutable, SHA-256 audit logs |
