# 🗺️ PathGuard: Project & AI Roadmap
## *The Journey to Global Financial Transparency*

PathGuard's development follows a rigorous phased approach, transitioning from a local-first security tool to a globally integrated verification ecosystem.

---

### 🚀 Phase 1: Foundation & Local Shell (Months 1-2)
**Goal**: Establish the secure desktop environment and local AI reasoning.
- **Tauri Core**: Build the secure Rust-based shell for Windows 11.
- **Ollama Integration**: Link the UI to local `llama3.1:8b` for private risk analysis.
- **Secure Vault**: Implement SQLCipher for AES-256 local database encryption.
- **Visual Timeline**: Prototype the "Live Tracker" UI for transaction flow.

---

### 🧠 Phase 2: Local ML Anomaly Engine (Months 3-4)
**Goal**: Embed numerical fraud detection directly into the client.
- **ONNX Runtime**: Integrate lightning-fast fraud scoring models into the Tauri binary.
- **Pattern Learning**: Implement local continuous learning to adapt to user-specific habits.
- **Incident Rerouting**: Pilot the "Bank Outage" detection and resolution workflows.
- **TPM Signing**: Leverage Windows Hardware-Backed keys (TPM) for approval signing.

---

### 🌊 Phase 3: Institutional Integration (Months 5-8)
**Goal**: Connect to the global financial data rails.
- **Secure WebSockets**: Implement TLS 1.3 encrypted streams for real-time ingestion.
- **Identity Middleware**: Deploy OAuth2/OIDC integration for institutional auth.
- **Immutable Ledger**: Launch the Merkle chained governance ledger for auditability.
- **Regulator Portal**: Beta release of the forensic dashboard for financial authorities.

---

### 🌐 Phase 4: Decentralized Verification (Months 9-12)
**Goal**: Extend trust through Web3 and community validation.
- **$GUARD Token**: Launch the incentivized verification ecosystem.
- **Federated Learning**: Implement privacy-preserving model updates across the network.
- **Disaster Mode**: Scale resilient routing for high-chaos humanitarian corridors.
- **SDK Ecosystem**: Open the PathGuard Sentinel API to third-party developers.

---

### 🧪 Future Vision: Autonomous Compliance
Beyond Phase 4, PathGuard aims to remove human lag from global finance. By utilizing sentient AI agents to monitor cross-border flows, we reduce AML investigation times by >90% while maintaining absolute cryptographic certainty.
